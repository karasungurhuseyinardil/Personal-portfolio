## 🇬🇧 English

This project is the personal portfolio website of **Hüseyin Ardıl Karasungur**.
It is designed with a modern, fast, and fully responsive architecture to professionally showcase a software developer’s skills, experience, and projects.

### 🚀 Features

* **Modern & Aesthetic Design**
  Premium user experience with glassmorphism, dark mode support, and smooth animations (including 3D tilt effects).

* **Advanced Multilingual System (EN/TR)**
  All content, placeholders, and metadata are dynamically managed using a structured dictionary system.

* **Dark / Light Mode**
  Dynamic theme switching with `localStorage` persistence.

* **Responsive Design**
  Fully optimized layout for mobile, tablet, and desktop devices.

* **PWA (Progressive Web App)**
  Offline support and installable application experience (`sw.js`).

* **Contact Form Integration**
  Email sending via EmailJS SDK with instant feedback (toast notifications).

* **GitHub Integration**
  Dynamic GitHub statistics using GitHub API.

* **Rich UI/UX Experience**
  Canvas-based interactive background, custom cursor, loader screen, and scroll reveal animations.

## 🇹🇷 Türkçe

Bu proje, **Hüseyin Ardıl Karasungur**'un kişisel portfolyo web sitesidir.
Modern, hızlı ve tamamen duyarlı (responsive) bir yapıyla geliştirilmiş olup, bir yazılım geliştiricinin yeteneklerini, deneyimlerini ve projelerini profesyonel bir şekilde sergilemeyi amaçlar.

### 🚀 Özellikler

* **Modern & Estetik Tasarım**
  Glassmorphism, karanlık mod ve akıcı animasyonlar (3D tilt efekti dahil) ile premium kullanıcı deneyimi.

* **Gelişmiş Çoklu Dil Sistemi (EN/TR)**
  Tüm içerikler, placeholder'lar ve meta veriler dinamik sözlük yapısıyla yönetilmektedir.

* **Karanlık / Aydınlık Mod**
  `localStorage` ile kalıcı tema desteği.

* **Duyarlı Tasarım (Responsive)**
  Mobil, tablet ve masaüstü için optimize edilmiştir.

* **PWA (Progressive Web App)**
  Çevrimdışı çalışma ve uygulama olarak yüklenebilme (`sw.js`).
