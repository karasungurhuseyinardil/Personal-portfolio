# Hüseyin Ardıl Karasungur — Developer Portfolio

Bu proje, **Hüseyin Ardıl Karasungur**'un kişisel portfolyo web sitesidir. Modern, hızlı ve responsive bir tasarıma sahip olan bu site, bir yazılım geliştiricinin yeteneklerini, deneyimlerini ve projelerini sergilemek amacıyla geliştirilmiştir.



## 🚀 Özellikler

- **Modern & Estetik Tasarım**: Glassmorphism, karanlık mod desteği ve akıcı animasyonlar (3D Tilt efekti dahil) ile premium bir kullanıcı deneyimi.
- **Gelişmiş Çoklu Dil Sistemi (EN/TR)**: Tüm içerik, placeholders ve meta veriler dahil olmak üzere dinamik bir sözlük yapısıyla hem İngilizce hem de Türkçe olarak sunulmaktadır.
- **Karanlık/Aydınlık Mod**: Kullanıcı tercihine göre değişen dinamik tema desteği ve `localStorage` kalıcılığı.
- **Duyarlı Tasarım (Responsive)**: Mobil, tablet ve masaüstü cihazlar için özel olarak optimize edilmiş sayfa düzeni ve carousel bileşenleri.
- **PWA (Progressive Web App) Desteği**: Çevrimdışı çalışma kabiliyeti ve uygulama olarak yüklenebilme özelliği (`sw.js`).
- **İletişim Formu Entegrasyonu**: EmailJS SDK'sı kullanılarak doğrudan e-posta gönderimi ve anlık bildirimler (toast notification).
- **GitHub Entegrasyonu**: GitHub API kullanılarak çekilen dinamik kullanıcı istatistikleri.
- **Zengin Görsel Deneyim**: Canvas tabanlı interaktif arka plan, özel imleç (cursor), loader ekranı ve scroll reveal efektleri.

## 🛠️ Kullanılan Teknolojiler

- **Core**: HTML5, CSS3 (Vanilla & Modern CSS Variables), JavaScript (ES6+, Async/Await)
- **Kütüphaneler & Servisler**:
    - **EmailJS SDK**: İletişim formu iletimi.
    - **FontAwesome 6**: Vektörel ikonlar.
    - **Google Fonts**: Inter ve Fira Code tipografisi.
    - **GitHub API**: Canlı repo verileri.
- **Teknik Detaylar**:
    - Intersection Observer API (Scroll Reveal & Counters)
    - Service Worker (PWA)
    - 3D Mouse Tilt Animasyonları
    - CSS Grid & Flexbox Mimarisi
